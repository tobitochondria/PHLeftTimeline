# Philippine Left Timeline Changelog

## Version 3 Updates

This release introduces several new organizations and updates descriptions across all major eras to reflect the broader scope of the Philippine Left. Reference options and additional sources were integrated via data parsing based on the Version 3 prompt. Note: Raw citations/sources were intentionally kept out of the parsed descriptions per Option B in the setup phase.

### Notable Modifications & Fixes
*   **Unión de Impresores de Filipinas (UIF)**: Date formally set to "November 1901 / 30 December 1901".
*   **Union de Tabaqueros de Filipinas (UTF)**: Now explicitly named with abbreviation mapping matching "Union de Tabaqueros de Filipinas (UTF)".
*   **Congreso Obrero de Filipinas (COF)**: Fixed name representation.
*   **Kalipunang Pangbansa ng mga Magbubukid sa Pilipinas (KPMP)**: Appended abbreviation string.
*   **Katipunan ng mga Anakpawis sa Pilipinas (KAP)**: Date updated from "1929" to "May 1929".
*   **Congress of Labor Organizations (CLO)**: Date officially registered as "March 1945 / July 1945".
*   **Makabayang Kilusan ng Bagong Kababaihan (MAKIBAKA)**: Minor structural tweaks in definition mapping.
*   **Panday Sining**: Added context noting its 1964 origin and later 2016 re-establishment as an artists-led mass organization.
